Java EE Application Server

The easiest way to use Weld is with a Java EE Application Server. Weld is used by the following application servers:

  • WildFly
  • JBoss Enterprise Application Platform 6
  • JBoss AS 7
  • GlassFish
  • Oracle WebLogic Server
  • WebSphere Application Server Liberty Profile

Maven

  • CDI API
  • Weld Core
  • Weld Servlet ( Core , Shaded jar with dependencies Weld 2 , Weld 3 )
  • Weld SE ( Core , Shaded jar with dependencies Weld 2 , Weld 3 )
  • Weld OSGi bundle
