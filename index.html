Weld is the reference implementation for CDI: Contexts and Dependency Injection for the Java EE platform. The CDI specification is a part of the Java EE platform, ie. any Java EE-compliant application server must provide support for CDI. Weld is integrated into many application servers, e.g. WildFly 8 and Oracle GlassFish 4.x.

Weld is available under the Apache License 2.0.
